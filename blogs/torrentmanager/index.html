    TorrentManager 说明文档

    shanke 2021/2/6 pttorrentunraid

    说明

    TorrentManager 开源项目介绍说明。

    注意：

    2021-03-05：项目已发布0.1.0版本，简化配置，优化操作。0.09以及之前版本在升级后，请务必修改下载器中下载目录映射（不使用链接或移动功能可忽略）。

    # 一、介绍

    开源地址：TorrentManager (gitee.com) (opens new window)

    Docker Hub：shanke520/torrentmanager (Docker Hub) (opens new window)

    因为玩PT，所以做了这个系统。主要是为了实现两点个人需求：

    1. 将种子信息维护起来，方便查看种子来源和分类等
    2. 种子下载完后，将下载的文件硬链接到Jellyfin的媒体库目录下。Jellyfin刮削修改的是硬链接后的文件，不会影响原文件做种了

    本人的nas系统是unraid，所以下载工具transmission和媒体服务jellyfin都是docker方式的。TorrentManager也基于docker进行部署。

    # 二、软件架构

    系统基于B/S结构，使用java开发。

    # 三、安装教程

    ​ nas系统：unraid

    # 2. 安装说明

    镜像地址：

    shanke520/torrentmanager [Docker Hub] (opens new window)

    配置：

    Host Container 说明
    Port 6699 6699 web ui 端口
    Path /mnt/user/appdata/torrentmanager /config TorrentManager的配置存放位置
    Path /mnt/user/mydata /manage 管理目录，transmittion下载目录和jellyfin媒体库目录共同的上级目录

    # 四、使用说明 - 基本配置

    说明的顺序：先是进行基础的一些配置，再是如何使用。

    # 1. 登录

    默认密码：password。

    进入系统后可进行修改，基本配置-修改密码。

    # 2. 基本配置

    • 修改密码
    • 目录映射，Docker版本无需配置（目前只提供了Docker版本）。

    # 3. 下载器

    配置Transmission连接信息，推送种子到transmission下载。

    参数 值 说明
    服务器名称 transmission 填写自己容易辨识的名称
    服务器类型 Transmission 目前只支持Transmission
    服务器地址 http://192.168.88.22:9091 Transmission地址，确保TorrentManager能访问到
    登录名 admin Transmission用户名
    登录密码 admin Transmission用户密码
    下载目录映射 原始目录：/downloads/compelete transmission下载目录，留空且连接信息配置正确，系统会自动获取
    映射目录：/manage/downloads/compelete 系统可识别的目录，选择原始目录所对应的目录
    描述

    关于下载目录映射的详细说明：

    transmission安装配置：

    Host Container 说明
    Path /mnt/user/mydata/downloads /downloads transmittion下载目录

    TorrentManager安装配置：

    Host Container 说明
    Path /mnt/user/mydata /manage TorrentManager管理目录

    如上配置中，假如下载一个a.mp4文件，对于宿主机、Transmission、TorrentManager三者各自环境中的路径为：

    • 宿主机：/mnt/user/mydata/downloads/complete/a.mp4

    • Transmission：/downloads/complete/a.mp4

    • TorrentManager：/manage/downloads/complete/a.mp4

    下载目录映射的作用就是让TorrentManager能够找到下载的文件。方便进行链接或移动到其他位置。

    # 4. 站点配置

    进行站点配置后，可为种子选定站点。

    • 方便种子按站点进行查看
    • 站点的目录影响种子文件的下载位置，格式为：【基础下载目录】/【站点目录】/【分类目录】

    例如，添加一个种子，站点为【比特校园】，分类为【电影】。

    【比特校园】的目录是BTSCHOOL，【电影】的目录是Movies，则下载文件的完整路径为：

    /mnt/user/mydata/downloads/BTSCHOOL/Movies/xxxxxx.mp4

    # 5. 分类配置

    进行分类配置后，可为种子选定分类。

    • 方便种子按分配进行查看
    • 分类的目录影响种子文件的下载位置，格式为：【基础下载目录】/【站点目录】/【分类目录】

    例如，添加一个种子，站点为【比特校园】，分类为【连续剧】。

    【比特校园】的目录是BTSCHOOL，【连续剧】的目录是Movies，则下载文件的完整路径为：

    /mnt/user/mydata/downloads/BTSCHOOL/TV-Series/xxxxxx.mp4

    # 五、使用说明 - 种子管理

    种子管理是最主要的功能，所以单独说明。

    # 1. 界面介绍

    1. 筛选
    2. 种子列表
    3. 添加、推送下载种子
    4. 从下载器同步种子信息
    5. 种子详情，查看种子基本信息和下载状态，未推送下载的种子进行推送，修改下载位置，将下载文件链接到媒体库
    6. 种子编辑
    7. 种子删除

    # 2. 添加一个种子进行下载

    点击(3)，开始往TorrentManager添加一个种子。

    # 基本信息

    参数 说明
    种子名称 自己进行命名，因种子自带信息中的名称很多都是英文的。也可以不填，推送成功后会使用种子自带信息中的名称。
    下载链接 磁力链接或者种子下载链接，Transmission支持即可。
    所属站点 从哪个站点获取的种子。推送下载时会在下载路径中添加站点目录，如不需要，可以在站点配置时把目录名称留空。
    所属分类 种子所属的分类,。推送下载时会在下载路径中添加分类目录，如不需要，可以在分类配置时把目录名称留空。
    描述 填写一些备注信息。

    # 下载配置

    选择需要推送到哪个下载器进行下载。可暂时不选择，保存后，可在种子详情再进行推送下载。

    # 3. 链接或移动到新的位置

    # Step1. 点击(5)，查看种子详情。

    # Step2. 查看下载信息中种子的完成情况，一般通过状态为Seeding判断，表示已经下载完成在做种了。点击链接或移动到新的位置。

    # Step3. 原本目录：文件原来存放的目录。

    # Step4. 目标目录：需要链接或移动的目标目录，支持选择和自定义。

    举例：

    选择目录为/manage/video/tv/纪录片，但需要片名目录和季目录，则手动进行添加为/manage/video/tv/纪录片/小小世界/Season 1，执行时不存在的目录会自动创建。

    # Step5. 文件名称替换：批量替换文件名。

    举例：

    列表中的文件名Tiny.World.S01.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT[EliteT]/Tiny.World.S01E01.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT.mkv，带有路径即含有目录，我已经在目标目录指定好了小小世界/Season 1，所以将Tiny.World.S01.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT[EliteT]/替换为空。

    当然，你也可以在列表中单个修改，添加或删除目录，执行时不存在的目录会自动创建。

    # Step6. 执行硬链接或移动。

    • 硬链接：点击进行硬链接操作，这样目标目录的文件让媒体软件刮削，不会影响原文件，不影响做种。

    • 移动：移动会影响做种，请谨慎操作。移动前系统会停止种子下载，移动完成后，下载的文件将被移动到目标目录下，请自行处理种子。

    举例：

    结合上面的步骤，硬链接或移动后的文件完整路径为:

    /manage/video/tv/纪录片/小小世界/Season 1/Tiny.World.S01E01.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT.mkv

    /manage/video/tv/纪录片/小小世界/Season 1/Tiny.World.S01E02.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT.mkv

    /manage/video/tv/纪录片/小小世界/Season 1/Tiny.World.S01E03.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT.mkv

    /manage/video/tv/纪录片/小小世界/Season 1/Tiny.World.S01E04.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT.mkv

    /manage/video/tv/纪录片/小小世界/Season 1/Tiny.World.S01E05.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT.mkv

    /manage/video/tv/纪录片/小小世界/Season 1/Tiny.World.S01E05.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT.mkv

    /manage/video/tv/纪录片/小小世界/Season 1/Tiny.World.S01.HDR.2160p.WEB-DL.DDP5.1.H.265-ROCCaT.nfo

    有些种子下载后文件中已包含刮削文件nfo、图片等，如不需要，只勾选视频文件即可。
