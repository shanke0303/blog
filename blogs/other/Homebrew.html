    HomeBrew 使用说明

    # 概要

    brew --version
// command：参考下文中的命令
brew command [--verbose|-v] [options] [formula] ...
    1
    2
    3

    # 必要命令

    完整的命令列表，请参考章节：全部命令。

    许多命令使用时添加--version或者--debug可以输出额外的调试信息。注意只能跟在命令后面。

    # 安装formula

    brew install formula
    1

    formula通常是需要安装的formula的名称，但是也有其它的语法，请参考章节：指定FORMULAE。

    # 卸载formula

    brew uninstall formula
    1

    # 查看已安装的formula

    brew list
    1

    # 查询

    brew search [text|/text/]
    1

    text是按名称查询cask和formula，/text/表示以正则表达式进行查询。查询的仓库为网络上的homebrew/core和homebrew/cask，如果不加text或者/text/使用search命令则列出本地所有可用的formula。

    实测不加搜索词使用命令在当前版本已经不支持，列出本地所有可用的formula使用brew formulae代替。

    % brew search       
Error: Calling `brew search` with no arguments to output formulae is disabled! Use `brew formulae` instead.
    1
    2

    # 全部命令

    # analytics [subcommand]

    管理Homebrew对用户行为的匿名合计分析。更多请参考官方相关文档：https://docs.brew.sh/Analytics (opens new window)。

    查看分析的状态：

    brew analytics [state]
    1

    打开或关闭分析：

    brew analytics (on|off)
    1

    重新生成分析时用到的UUID：

    brew analytics regenerate-uuid
    1

    # autoremove [--dry-run]

    卸载一些formula，这些formula只是其它formula不再需要的依赖。

    • -n，--dry-run：仅列出哪些会被卸载，不会真正执行卸载。

    # casks

    列出所有本地可安装的cask。

    # cleanup [options] [formula|cask ...]

    1. 移除所有formula和cask的已过时锁文件、过期下载，移除已安装的旧版本formula。

      [formula|cask ...]：指定formula或cask。

    2. 移除超过120天的所有下载，超期天数通过HOMEBREW_CLEANUP_MAX_AGE_DAYS配置调整。

    • --prune：指定超期天数来移除缓存文件，移除所有：--prune=all。
    • -n，--dry-run：仅列出哪些会被移除，不会真正执行移除。
    • -s：清理缓存，包括最新版本的下载。已安装的formula或cask的下载不会被删除，如果要删除则使用：rm -rf "$(brew --cache)"。
    • --prune-prefix：仅操作包含指定前缀的符号链接和目录，而不删除其他文件。

    # commands [--quiet] [--include-aliases]

    列出内置的和外部的命令。

    • -q，--quiet：不显示命令的类别，仅列出命令的名称。
    • --include-aliases：包括内部命令的别名。

    # leaves [--installed-on-request] [--installed-as-dependency]

    列出不是作为其他formula的依赖所安装的formulae。

    ps: 经常在brew uninstall后使用，因为卸载不会卸载其相关依赖。

    • -r, --installed-on-request: 只查询手动安装的。
    • -p, --installed-as-dependency: 只查询作为依赖被安装的。

    参考文档：https://docs.brew.sh/Manpage (opens new window)
